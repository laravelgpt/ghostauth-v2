# Security Policy

## Reporting Security Issues

Please report security vulnerabilities to laravelgpt@example.com or through the GitHub Security Advisory process.

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Security Practices

This package follows security best practices including defense-in-depth, input validation, output encoding, and secure defaults.
